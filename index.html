Neural network model training for Long Short Term Memory using Tensorflow, Anaconda (Python), Muisc21 and generating music.

GitHub link:https://github.com/T1794584785/LSTM-RNN_MUSIC-GENERATION

First install the Anaconda environment, note: Python 3.6-3.7 is required for Tensorflow deployment.

There are three big and important scripts that build the framework of the algorithm: "Preprocessing", "Training the model", and "Generating the music".
